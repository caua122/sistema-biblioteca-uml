# sistema-biblioteca-uml

Sistema de Biblioteca

O Sistema de Biblioteca tem como objetivo gerenciar o controle de livros, usuários e empréstimos de forma eficiente. Ele permite o cadastro de livros, incluindo informações como título, autor, categoria e disponibilidade.

Os usuários podem se cadastrar no sistema para realizar empréstimos de livros. Cada empréstimo possui uma data de retirada e uma data de devolução, permitindo o controle de prazos e possíveis atrasos.

O sistema também possibilita:

Consulta de livros disponíveis
Controle de devoluções
Histórico de empréstimos dos usuários
Aplicação de penalidades em caso de atraso

Além disso, há diferentes tipos de usuários, como administradores (responsáveis pelo gerenciamento do sistema) e leitores (que utilizam o serviço de empréstimo).

O objetivo principal é organizar e automatizar os processos de uma biblioteca, garantindo melhor controle do acervo e facilitando o acesso dos usuários aos livros.
